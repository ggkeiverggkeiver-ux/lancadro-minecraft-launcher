# lancadro-minecraft-launcher
lancador is my wondered minecraft launcher and easy and 🚀 Installation Guide (English)
To run Lanzador correctly on macOS, you must download both ZIP files. Follow these steps:
Download both files: Download Lanzader-files.zip and Lanzader.app.zip.
Extract both: Unzip (double-click) both files in the same folder (e.g., your Downloads or Desktop).
Combine if necessary: Ensure all the library files (like .dylib and Launcher.js) are in the same directory where Lanzador.app is located.
How to Open:
Since this is an indie project, macOS might show a warning about an "unverified developer."
To bypass this: Right-click (or Control-click) Lanzador.app and select Open. Then click Open again in the pop-up window.
Enjoy playing Minecraft for free! 🎮
